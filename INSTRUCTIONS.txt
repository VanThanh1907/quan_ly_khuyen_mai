STEP-BY-STEP GUIDE:

Step 1: START MONGODB
----------------------
  Option A: Using MongoDB Compass
    - Open MongoDB Compass application
    - Click "Connect" (default: localhost:27017)
  
  Option B: Using Command Line
    - Double-click: start-mongodb.bat
    - OR open terminal and run: mongod
    - Keep the window open!

Step 2: TEST MONGODB CONNECTION
--------------------------------
  - Double-click: test-mongodb.bat
  - You should see: "Connection test PASSED!"
  - If failed, make sure Step 1 is done correctly

Step 3: INSTALL DEPENDENCIES (First time only)
-----------------------------------------------
  - Double-click: setup.bat
  - Wait for installation to complete (2-5 minutes)
  - This creates sample data automatically

Step 4: START THE APPLICATION
------------------------------
  - Double-click: run-all.bat
  - Browser will open automatically
  - You'll see 2 new windows (Backend + Frontend)

Step 5: LOGIN AND USE
----------------------
  - Wait for browser to open at: http://localhost:3000
  - Login with:
    Admin: admin / admin123
    User:  user / user123

AVAILABLE SCRIPTS:

test-mongodb.bat    - Test MongoDB connection
setup.bat           - Install dependencies + seed data
reset-project.bat   - Clean install everything
stop-all.bat        - Stop all servers

========================================
TROUBLESHOOTING:
========================================

Problem: "Cannot connect to MongoDB"
Solution: 
  1. Run: start-mongodb.bat
  2. Run: test-mongodb.bat
  3. If still fails, restart computer

Problem: "Port already in use"
Solution:
  1. Run: stop-all.bat
  2. Run: run-all.bat

Problem: "Module not found"
Solution:
  1. Run: reset-project.bat
  2. Run: run-all.bat

Problem: Application won't start
Solution:
  1. Check MongoDB: test-mongodb.bat
  2. Reset project: reset-project.bat
  3. Start again: run-all.bat
